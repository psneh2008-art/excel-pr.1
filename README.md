# Fundamental Booster – Excel Project

## Overview
This project file **PR. 1 Fundamental Booster (1).xlsx** is designed to strengthen fundamental Excel skills through multiple practical datasets and tasks.

The workbook contains **4 sheets**, each focused on a different real‑world scenario such as education, sales, and employee management.

---

## File Structure

### 1. Project Instructions
- Contains overall guidelines and tasks to be performed
- Explains what formulas, functions, or analysis are expected
- Acts as the reference sheet for the entire project

### 2. Students Grade
- Student academic data
- Used for practicing:
  - Percentage calculation
  - Grade logic (IF, nested IF)
  - Average, Max, Min
  - Pass / Fail analysis

### 3. Sales Data
- Product‑wise or date‑wise sales information
- Used for:
  - Total sales calculation
  - Revenue analysis
  - SUM, AVERAGE, COUNT functions
  - Basic performance insights

### 4. Employee Data
- Employee‑related records
- Used for:
  - Salary calculations
  - Department‑wise analysis
  - Conditional formulas
  - Data organization and formatting

---

## Skills Covered
- Basic to intermediate Excel formulas
- Logical functions (IF, AND, OR)
- Mathematical functions (SUM, AVERAGE)
- Data formatting
- Real‑world data handling
- Analytical thinking

---

## How to Use
1. Open the Excel file in Microsoft Excel or compatible software.
2. Start with **Project Instructions** sheet.
3. Complete tasks sheet‑by‑sheet.
4. Verify results using formulas, not manual calculations.

---

## Requirements
- Microsoft Excel 2016 or later (recommended)
- Basic understanding of Excel interface
